# 信息面板模块使用手册

## 1. 模块概述

信息面板是后台管理系统的综合数据展示模块，提供维修项目中工作人员状态、物料库存和维修器具使用情况的实时监控。通过可视化方式帮助管理员快速掌握资源分配和使用状况。

**主要功能**：
- 工作人员状态实时监控
- 班组工单进度查看
- 物料库存使用看板
- 维修器具状态管理
- 工人任务日历查看

---

## 2. 界面布局

### 2.1 整体布局

信息面板采用左右分栏布局：

| 区域 | 位置 | 功能 |
|------|------|------|
| 时间周期选择器 | 页面顶部 | 设置数据查询的时间范围 |
| 左侧面板 | 左半部分 | 工作人员状态/班组视图 |
| 右侧面板 | 右半部分 | 物料库存看板 + 维修器具看板 |

### 2.2 时间周期选择器

位于页面顶部，用于设置数据查询的时间范围：

- **开始天数**：项目第几天开始（默认第1天）
- **开始时间**：当天的开始时间（08:00-20:00，每半小时一格）
- **结束天数**：项目第几天结束（默认最后一天）
- **结束时间**：当天的结束时间（08:00-20:00，每半小时一格）

时间范围变更后，所有数据会自动刷新。

## 3. 核心功能

### 3.1 工作人员状态监控

#### 3.1.1 视图切换

顶部提供「工作人员」和「班组」两个视图切换按钮：

- **工作人员视图**：按工人维度展示状态
- **班组视图**：按班组维度展示工单进度

#### 3.1.2 工作人员视图

**筛选功能**：
- 搜索框：按工人姓名搜索
- 状态筛选：全部、工作中、空闲中、已完成

**表格信息**：
- 工人姓名
- 角色（如安全员、钳工等）
- 当前状态
- 任务列表（可折叠展开）
- 「点击查看任务日历」入口

**操作**：点击「点击查看任务日历」可打开工人详细任务日历弹窗

#### 3.1.3 班组视图

**筛选功能**：
- 按设备筛选：选择特定设备查看相关班组进度
- 按工序筛选：选择特定工序查看班组分配情况

**展示内容**：
- 工单进度可视化
- 状态分布统计
- 工人角色分组展示

### 3.2 工人任务日历

**打开方式**：在工作人员视图中，点击工人卡片或「点击查看任务日历」提示

**日历弹窗包含**：

| 区域 | 功能 |
|------|------|
| 标题 | 显示"[工人姓名] 的任务课表" |
| 周切换 | 支持上一周/下一周切换，显示当前周数/总周数 |
| 设备筛选 | 可按设备筛选任务 |
| 快捷周选择 | 显示所有周数按钮，可直接跳转到指定周 |
| 时间网格 | 7天×24小时的课表网格（08:00-20:00，每半小时一格） |
| 任务卡片 | 显示任务名称、设备名称、时间范围 |

**使用技巧**：
- 通过周切换按钮查看不同时间段的任务安排
- 使用设备筛选快速定位特定设备的任务
- 点击快捷周按钮直接跳转到目标周

### 3.3 物料库存使用看板

**功能概述**：以环形图形式展示各类物料的库存使用情况

**展示内容**：

每个物料卡片包含：
- **物料名称**：显示物料名称
- **环形图**：可视化库存使用比例
  - 绿色：剩余库存（计划使用后）
  - 黄色：计划使用量
  - 红色：已使用量
- **库存数量**：显示当前库存数量和单位
- **计划使用**：显示计划使用数量
- **已使用**：显示实际已使用数量

**库存状态标识**：
- 🟢 库存正常：所有物料使用率低于80%
- 🔴 库存预警：存在物料使用率达到或超过80%

**筛选功能**：顶部提供任务筛选下拉框，可选择特定任务查看物料使用情况

### 3.4 维修器具使用状态看板

**功能概述**：以表格形式展示所有维修器具的使用状态

**表格列**：

| 列名 | 说明 |
|------|------|
| 器具名称 | 维修器具名称 |
| 器具类型 | 设备分类（起重设备、运输设备、焊接设备等） |
| 使用状态 | 可用（绿色）/ 占用（红色） |
| 使用任务 | 占用时显示正在使用的任务名称 |

**筛选功能**：顶部提供按类型筛选下拉框，可选择特定类型查看

**类型分类**：
- 起重设备
- 运输设备
- 焊接设备
- 通风设备
- 加热设备

## 4. 使用流程

### 4.1 日常监控流程

1. **设置时间范围**：在时间周期选择器中设置查询的时间段
2. **查看人员状态**：在左侧面板切换「工作人员」视图，查看工人当前状态
3. **查看班组进度**：切换到「班组」视图，查看各班组的工单进度
4. **检查物料库存**：在右侧上方查看物料库存看板，注意库存预警
5. **检查器具状态**：在右侧下方查看维修器具使用状态，确认可用资源

### 4.2 详细任务查看

1. 在工作人员视图中找到目标工人
2. 点击「点击查看任务日历」
3. 在日历弹窗中查看该工人的详细任务安排
4. 使用周切换和设备筛选定位特定任务

### 4.3 资源调配参考

1. 通过物料库存看板识别库存紧张的物料
2. 通过维修器具看板识别可用的设备资源
3. 通过工作人员状态安排合适的人员

## 5. 使用技巧

### 5.1 快速定位资源

- 使用时间周期选择器缩小查询范围，聚焦特定时间段
- 在工作人员视图中使用搜索框快速找到目标工人
- 在班组视图中按设备筛选查看特定设备的班组安排

### 5.2 库存预警处理

- 当库存状态显示"库存预警"时，及时检查相关物料的库存情况
- 对比计划使用量和已使用量，评估是否需要补充物料

### 5.3 器具资源优化

- 优先安排使用状态为"可用"的器具
- 对于占用中的器具，查看使用任务了解预计释放时间

### 5.4 任务冲突排查

- 通过工人任务日历查看工人的时间安排
- 检查同一时间段是否存在任务冲突

## 6. 常见问题

**Q1：为什么某些工人没有任务？**

A1：工人可能处于空闲状态，或者当前查询的时间范围内没有分配任务。尝试调整时间周期选择器查看其他时间段。

**Q2：物料库存预警是什么意思？**

A2：当物料的实际使用率达到或超过80%时，系统会显示"库存预警"，提醒管理员注意该物料可能即将耗尽，需要及时补充。

**Q3：维修器具显示"占用"但没有使用任务？**

A3：可能是数据同步延迟，刷新页面后重新查看。如果问题持续存在，请检查后台数据。

**Q4：为什么工人日历中没有任务显示？**

A4：确认时间周期选择器设置的范围是否包含该工人的任务时间，尝试扩大时间范围或切换周数查看。

## 7. 技术说明

- **前端框架**：Vue3 + Element Plus
- **数据接口**：
  - 工作人员状态：`/api/worker-status`
  - 工单信息：`/api/work-orders`
  - 物料库存：`/api/material-inventory`
  - 维修器具：`/api/maintenance-tools`
- **可视化**：SVG环形图自定义绘制
- **响应式设计**：支持不同屏幕尺寸的自适应布局
