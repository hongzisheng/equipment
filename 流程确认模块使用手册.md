# 流程确认模块使用手册

## 1. 模块概述

流程确认模块是后台管理系统中的核心审批模块，用于对设备维修工序进行全流程状态管理和审批操作。管理员可以通过此模块查看所有工序的状态、审批待处理的工序、驳回不合规的流程，以及取消已创建的工序。

**主要功能**：
- 工序状态实时监控
- 多维度设备筛选
- 单条/批量审批确认
- 工序详情查看与图片上传
- 操作日志追溯

---

## 2. 界面布局

### 2.1 整体布局

流程确认页面采用卡片式布局，主要包含三个区域：

| 区域 | 位置 | 功能 |
|------|------|------|
| 标题栏 | 页面顶部 | 显示模块名称和批量操作按钮 |
| 筛选区域 | 标题栏下方 | 按设备属性和状态进行筛选 |
| 工序列表 | 页面主体 | 表格展示所有工序信息 |

### 2.2 标题栏

- **左侧**：模块图标 + "流程确认" 标题
- **右侧**：「批量确认」按钮（仅当有待审批任务时可用）

### 2.3 筛选区域

筛选区域提供四级联动筛选：

1. **设备种类**：按设备大类筛选（如空冷器、泵类等），显示每种的数量统计
2. **设备类型**：在选中种类下，按具体类型筛选，显示数量统计
3. **设备实例**：在选中类型下，按具体设备实例筛选，支持搜索
4. **工单状态**：按工序当前状态筛选，显示各状态数量

筛选区域底部显示当前筛选结果的统计信息，以及「清除所有筛选」按钮。

### 2.4 工序列表

表格包含以下列：

| 列名 | 说明 | 特殊功能 |
|------|------|----------|
| 序号 | 工序编号 | 支持排序 |
| 设备名称 | 所属设备 | 图标标识 |
| 设备种类 | 设备分类 | - |
| 设备类型 | 设备具体类型 | - |
| 工序名称 | 工序名称 | 里程碑标记 |
| 状态 | 当前状态 | 彩色状态标签 |
| 责任人 | 指派的工人 | 多位工人显示 |
| 预计时长 | 预计工期 | 以天为单位 |
| 开始时间 | 计划开始时间 | - |
| 结束时间 | 计划结束时间 | - |
| 操作 | 操作按钮 | 详情、取消 |

**表格特性**：
- 不同状态显示不同背景色（进行中-蓝色、已驳回-红色、已完成-绿色等）
- 里程碑工序带有黄色标签标记
- 支持行点击查看详情

## 3. 核心功能

### 3.1 查看工序详情

**操作方式**：点击表格任意行或「详情」按钮

详情弹窗包含以下信息区域：

#### 3.1.1 设备信息
- 设备名称
- 设备种类
- 设备类型

#### 3.1.2 工序信息
- 工序名称
- 工序状态（标签显示）
- 责任人
- 预计时长
- 开始/结束时间

#### 3.1.3 工序描述
显示工序的详细说明（如有）

#### 3.1.4 上传图片
- 已上传图片预览（可点击放大）
- 支持更换或上传新图片
- 支持格式：jpg、png、gif、bmp、webp
- 单张图片限制：10MB

#### 3.1.5 操作日志
按时间顺序展示所有操作记录，包括：
- 操作类型标签（确认通过、驳回、取消、状态变更、上传图片）
- 状态流转（原状态 → 新状态）
- 操作时间
- 审批意见或描述

#### 3.1.6 物料需求
列出该工序所需的物料及其数量和单位

#### 3.1.7 工具需求
列出该工序所需的工具及其数量和单位

#### 3.1.8 工序流程
以时间线形式展示同一设备下的所有工序及其当前状态，点击可切换查看其他工序

#### 3.1.9 审核意见
审批时填写意见的输入框，支持200字以内

### 3.2 确认工序

**适用状态**：待工程师确认、待施工确认、待班组受理等非终态且非工人操作状态

**操作方式**：
1. 在详情弹窗中填写审核意见（可选）
2. 点击「确认」按钮
3. 在确认对话框中点击「确认」

确认后工序将进入下一状态，操作记录会写入日志。

### 3.3 驳回工序

**适用状态**：非终态、非待开始、非待施工回签状态

**操作方式**：
1. 在详情弹窗中点击「驳回」按钮
2. 在弹出的对话框中填写驳回原因（必填）
3. 点击「确定驳回」

驳回后工序退回上一状态，驳回原因会记录到日志中。

### 3.4 取消工序

**适用状态**：非已完成、非已取消状态

**操作方式**：
1. 在工序列表中点击「取消」按钮
2. 在确认对话框中点击「确定取消」

取消后工序状态变为"已取消"，不可恢复。

### 3.5 批量确认

**适用条件**：存在待审批任务（非终态且非工人操作状态）

**操作方式**：
1. 点击页面顶部的「批量确认」按钮
2. 在确认对话框中查看待确认任务数量
3. 点击「确认」按钮

系统会自动对所有待审批任务执行确认操作，统一填写"批量确认"作为审批意见。

## 4. 状态流转说明

### 4.1 管理员可操作的状态

| 状态 | 可执行操作 | 说明 |
|------|------------|------|
| 待工程师确认 | 确认/驳回 | 等待工程师审批 |
| 待施工确认 | 确认/驳回 | 等待施工审批 |
| 待班组受理 | 确认/驳回 | 等待班组接收 |
| 进行中 | 确认/驳回 | 工序执行中 |
| 已提交 | 确认/驳回 | 工人提交后等待审批 |
| 待工艺关闭 | 确认/驳回 | 等待工艺部门关闭 |
| 待设备部关闭 | 确认/驳回 | 等待设备部关闭 |

### 4.2 管理员不可操作的状态

| 状态 | 说明 |
|------|------|
| 待开始 | 工人操作，等待工人开始 |
| 待施工回签 | 工人操作，等待工人回签 |
| 已完成 | 终态，流程结束 |
| 已取消 | 终态，流程终止 |

### 4.3 状态颜色标识

| 颜色 | 状态 |
|------|------|
| 蓝色 | 进行中、待审批 |
| 红色 | 已驳回 |
| 绿色 | 已完成、已提交 |
| 橙色 | 暂停、待开始 |
| 紫色 | 已取消 |

## 5. 使用技巧

### 5.1 快速定位工序
- 使用设备种类→设备类型→设备实例的三级筛选快速定位到具体设备的工序
- 使用状态筛选快速找到待审批的任务

### 5.2 高效审批
- 对于大量待审批任务，使用「批量确认」功能提高效率
- 在详情弹窗中，可通过工序流程时间线快速切换查看同设备的其他工序

### 5.3 图片管理
- 上传的图片会自动预览，点击可查看大图
- 支持更换已上传的图片，旧图片会被覆盖

### 5.4 日志追溯
- 操作日志记录了完整的状态变更历史
- 可查看每次操作的审批意见和时间

## 6. 常见问题

**Q1：为什么某些工序没有确认/驳回按钮？**

A1：这些工序可能处于终态（已完成、已取消）或属于工人操作的状态（待开始、待施工回签），管理员无法对其进行审批操作。

**Q2：批量确认会影响哪些工序？**

A2：批量确认仅对当前列表中所有"待审批"状态的工序生效，终态和工人操作状态的工序不会被影响。

**Q3：取消后的工序还能恢复吗？**

A3：不能。取消操作是不可逆的，取消后的工序状态变为"已取消"，无法恢复到之前的状态。

**Q4：上传图片失败怎么办？**

A4：请检查图片格式（仅支持jpg/png/gif/bmp/webp）和大小（不超过10MB），确认后重新上传。

## 7. 技术说明

- **前端框架**：Vue3 + Element Plus
- **数据来源**：`/api/work-order-tasks` 接口
- **图片上传**：`/api/work-order-tasks/{id}/upload-image`
- **操作日志**：`/api/work-order-tasks/{id}/logs`
